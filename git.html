Git Commands - Полная шпаргалка

Все основные команды Git с пояснениями и примерами

💡 Подсказка: Все команды Git выполняются в терминале. Для копирования команды - кликните на неё.

📁 Основные команды

git init
Инициализировать новый Git репозиторий в текущей папке
# Создает скрытую папку .git
git clone [url]
Склонировать удаленный репозиторий на локальную машину
git clone https://github.com/user/repo.git
git status
Показать состояние рабочей директории и staging area
git log
Показать историю коммитов
git log --oneline # Краткий формат

📄 Работа с файлами

git add [file]
Добавить файл в staging area
git add index.html # Конкретный файл
git add .
Добавить ВСЕ измененные файлы в staging area
Внимание! Добавляет все файлы
git reset [file]
Убрать файл из staging area (отменить git add)
git rm [file]
Удалить файл из репозитория и рабочей директории
git mv [old] [new]
Переименовать или переместить файл

💾 Коммиты

git commit -m "сообщение"
Создать коммит с сообщением
git commit -m "Добавлена главная страница"
git commit -am "сообщение"
Добавить ВСЕ измененные файлы и сделать коммит
Упрощение git add . + git commit
git commit --amend
Изменить последний коммит (добавить файлы или изменить сообщение)
Осторожно! Изменяет историю
git show [commit]
Показать информацию о конкретном коммите

🌿 Ветки (Branches)

git branch
Показать все ветки (текущая выделена *)
git branch [name]
Создать новую ветку
git branch feature/login
git checkout [branch]
Переключиться на ветку
git checkout develop
git checkout -b [name]
Создать и переключиться на новую ветку
Популярно git checkout -b feature/payment
git merge [branch]
Влить указанную ветку в текущую
git merge feature/login
git branch -d [name]
Удалить ветку (безопасно)

☁️ Удаленные репозитории

git remote add [name] [url]
Добавить удаленный репозиторий
git remote add origin https://github.com/user/repo.git
git push [remote] [branch]
Отправить коммиты в удаленный репозиторий
git push origin main
git pull [remote] [branch]
Скачать изменения из удаленного репозитория и смержить
git pull origin develop
git fetch [remote]
Скачать изменения из удаленного репозитория (без мержа)
git remote -v
Показать все настроенные удаленные репозитории

↩️ Отмена изменений

git checkout -- [file]
Отменить изменения в файле (до последнего коммита)
git checkout -- script.js
git reset --hard [commit]
Откатить к определенному коммиту (удаляет последующие коммиты)
Опасно! git reset --hard HEAD~1
git revert [commit]
Создать новый коммит, отменяющий изменения указанного коммита
Безопасно Не переписывает историю
git clean -fd
Удалить неотслеживаемые файлы и директории
Осторожно! Файлы удаляются навсегда
⚠️ Внимание! Команды с Опасно! и Осторожно! могут привести к потере данных. Используйте их аккуратно!
