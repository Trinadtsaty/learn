🚀 Горячие клавиши VS Code

Для бывших пользователей PyCharm Community Edition

💡 Быстрый старт: Для автоматического создания HTML-документа введите ! и нажмите Tab

🎯 Навигация и редактирование

Ctrl + D
Выделить следующее вхождение текущего выделения
Ctrl + Shift + L
Выделить все вхождения текущего выделения
Alt + Click
Добавить курсор в указанное место
Ctrl + Alt + ↑/↓
Добавить курсор выше/ниже
Ctrl + /
Закомментировать/раскомментировать строку
Alt + Shift + ↑/↓
Скопировать строку вверх/вниз
Ctrl + Shift + K
Удалить строку

📊 Блоковое выделение Аналог PyCharm

Shift + Alt + Mouse drag
Блоковое выделение мышью (аналог Alt+Shift+Insert)
Ctrl + Shift + Alt + ↑/↓
Блоковое выделение стрелками
Ctrl + Shift + Alt + PgUp/PgDn
Блоковое выделение страницами
Shift + Alt + I
Добавить курсоры к концам всех выделенных строк
Ctrl + Shift + P
Открыть командную палитру → "Column Selection"

🔧 Git

Ctrl + Shift + G
Открыть панель Git
Ctrl + Enter
Commit (когда в поле сообщения коммита)
Ctrl + Shift + Enter
Push
⚠ Установи GitLens
Для коммита одной кнопкой как в PyCharm

📁 Файлы и поиск

Ctrl + N
Новый файл
Ctrl + O
Открыть файл
Ctrl + S
Сохранить
Ctrl + F
Поиск в файле
Ctrl + H
Замена в файле
Ctrl + Shift + F
Поиск по всем файлам

F12

Назад/вперед по истории навигации

Скрыть/показать боковую панель
